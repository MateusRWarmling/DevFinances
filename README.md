# DevFinances

<img src="https://i.imgur.com/lUivrUG.png" alt="exemplo imagem">

Aplicação para controlar finanças, com a possibilidade de adicionar, remover e listar transações financeiras, exibindo também a soma de entradas, saídas e do total. Fiz a aplicação com o intuito de treinar meus conhecimentos nos três pilares do desenvolvimento web(HTML, Javascript e CSS).

## ☕ Usando DevFinances

1. Clonar repositório
2. Abrir o arquivo index.html
